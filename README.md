# AI CSPM Overwatch

## Головне дослідницьке питання
**Чи здатні великі мовні моделі (LLM) ефективно виконувати роль контекстно-залежного хмарного аудитора, надійно аналізуючи результати CSPM-інструментів на основі архітектурних діаграм для відсіювання false positives та обґрунтованого коригування рівня ризику (NIST 800-53)?**

## Огляд проєкту
**Мета дослідження**: Порівняльний аналіз LLM у задачах контекстного відсіювання False Positives та валідації компенсуючих заходів безпеки, які не здатні зафіксувати стандартні CSPM-інструменти.

**Суть проєкту**: Дослідження здатності LLM аналізувати не просто окремі алерти CSPM, а також архітектурний контекст (в майбутньому із врахуванням коду конфігурації, але в цьому дослідженні я роблю припущення код не надається аудитору).

* **Аналітика та Dashboard**: Пайплайн розраховує узгодженість рішень (Decision Consistency) та відображає статистику у сучасному React застосунку (Recharts + Framer Motion).

## Бюджет та Витрати (Budget & Costs)

> [!WARNING]
> Запуск цього пайплайну вимагає фінансових витрат. Детальний прорахунок буде додано після завершення замірів, але варто враховувати наступне:
> - **AWS**: Створення ресурсів через Terraform є платним. Дещо може покриватися Free Tier, але загальна вартість залежатиме від кількості та часу існування конфігурацій до виконання `terraform destroy`.
> - **OpenRouter (LLM)**: Для аналізу результатів використовуються платні моделі. Бажано мати баланс більше $10 на акаунті OpenRouter, оскільки це знімає жорсткі ліміти швидкості (rate limits) на запити до платних моделей.
