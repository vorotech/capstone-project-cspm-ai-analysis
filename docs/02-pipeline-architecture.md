# Архітектура Пайплайну

Цей документ описує етапи роботи автоматизованого пайплайну.

## Етапи

1. **Git Checkout**: Клонування репозиторію `https://github.com/epam/ecc-aws-rulepack` та виділення необхідних `red` та `green` конфігурацій з папки `terraform`.
2. **Terraform Apply**: Локальний запуск вибраних тестових конфігурацій для створення AWS ресурсів.
3. **Аналіз CSPM**:
   - Запуск **Prowler** та збір результатів.
   - Запуск **Cloud Custodian** та збір результатів.
   - Збір результатів з **AWS Security Hub**.
4. **Terraform Destroy**: Видалення створених ресурсів для зачистки середовища.
5. **LLM Аналіз**: Обробка зібраних "сирих" результатів через 5 різних LLM (через OpenRouter API).
6. **Аналіз результатів**: Порівняння результатів моделей з еталоном (в Jupyter Notebook).
7. **Презентація**: Відображення даних за допомогою React + Recharts.
